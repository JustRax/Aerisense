# Aerisense
Aerisense is an Arduino project for personal air purifier



TOOLS AND TECHNOLOGIES USED:
Aruino IDE
Arduino UNO
Breadboard
Jumping Wires
MQ-125 Sensor
LEDS
Active Buzzer
Resistors
IRF520 MOSFET
Potentiometer
12v DC Fan
Saline PVC Pipe
Carbon Activated Foam Filter
9v Battery and Connectors
